# 🏦 Sistema Bancário Simples em Java

## 📌 Descrição

Este projeto consiste em uma aplicação em Java que simula o funcionamento básico de uma conta bancária.

O sistema permite criar uma conta com ou sem depósito inicial, além de realizar operações de depósito e saque. No caso do saque, é aplicada automaticamente uma taxa fixa sobre o valor retirado.

---

## 🚀 Funcionalidades

* Criação de conta bancária com:

  * Número da conta
  * Nome do titular
* Opção de depósito inicial
* Realização de depósitos
* Realização de saques com taxa fixa
* Exibição atualizada dos dados da conta

---

## 🧱 Estrutura do Projeto

O projeto é dividido em duas classes principais:

* `Banco` → Classe principal responsável pela execução do programa e interação com o usuário
* `conta1` → Classe responsável por representar a conta bancária e suas operações

---

## 🖥️ Tecnologias Utilizadas

* Java
* Programação Orientada a Objetos (POO)
* Entrada de dados com `Scanner`

---

## 📚 Conceitos Aplicados

* Encapsulamento (uso de atributos `private`)
* Construtores sobrecarregados
* Métodos para manipulação de saldo
